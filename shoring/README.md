The [Panopset](https://panopset.com) Java Maven parent project is shoring.
These are the Maven modules, not to be confused with the Java module names,
which are the same except with "com.panopset." prepended:


* [compat](compat/README.md), Panopset general Java utility classes.
* [desk](desk/README.md), Panopset JavaFX applications.
* [blackjackEngine](blackjackEngine/README.md),
  game engine for both the JavaFX and Android blackjack games.
* [flywheel](flywheel/README.md), freeform text template processor.
  It can be your secret, too.
* [web](web/README.md), [panopset.com](https://panopset.com) server.
* [fxapp](fxapp/README.md), Panopset JavaFX application foundations, including the standard menus.
* [joist](joist/README.md), contains the launchers module for the JavaFX application platform installers.
* [lowerclass](lowerclass/README.md), Java class and jar file analysis tool, useful for finding the least
  common JRE that your application dependencies may run on.

Here is the [javadoc](https://panopset.com/maven/index.html).
