# Desktop Applications

All Panopset desktop applications are available either through an all-in-one Java jar, or through a
platform installer.  The all-in-one jar, panopset.jar, requires that you have Java installed on your system.
The platform installer doesn't have that requirement, and will install launchers for you, per the standards
of your platform.

## Download

See the Panopset [download](https://panopset.com/download) page for panopset.jar
and/or installers available for selected x86 64-bit PC platforms.

## Version

To update the version:


* Add new version description to docs/vh.html.
* Update PV in deploy.properties.
* Run:


    ant version
